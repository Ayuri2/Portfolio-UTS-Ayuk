Tentang Saya

Ketahui mengenai profil dan diri saya

Saya Ni Putu Ayuk Riska Dewi, mahasiswi semester 5 yang sedang menempuh pendidikan Sarjana Sistem Informasi di Primakara University.

Ketertarikan saya adalah pada Database dan pengetahuan mengenai Manajemen Bisnis, dimana ilmu ini saya gunakan untuk membantu mengelola dan menganalisis perkembangan bisnis menggunakan teknologi berupa coding dan software. Selain itu, saya memiliki ketertarikan penunjang lainnya yaitu UI Design dengan menggunakan tools Figma.

Sejauh ini saya telah membuat sebanyak empat projek yang menggunakan database tools, graphic designing tools, dan Figma, yang saya cantumkan pada web ini.
